# Source Notes

Checked: 2026-07-18.

This is a local-origin workflow for preserving useful project knowledge and
archiving exact Codex tasks. No external retrospective or thread-management
workflow is copied.

On 2026-07-13, a self-archive attempt interrupted its own closeout turn before
the final result could be reported. The local design therefore requires a
separate controller, exact current-host task IDs, and archive as the target's
last mutation.

Key local preferences are manual invocation; one exact target, an exact list,
or an explicit exact-project sweep; no automatic age or count rule; Codex's
semantic judgment about documentation value; and native archive evidence.
