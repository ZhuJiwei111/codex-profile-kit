# Source Notes

Checked: 2026-07-18.

The design is informed by Superpowers v6.1.1
[`verification-before-completion`](https://github.com/obra/superpowers/blob/d884ae04edebef577e82ff7c4e143debd0bbec99/skills/verification-before-completion/SKILL.md)
at commit `d884ae04edebef577e82ff7c4e143debd0bbec99`, licensed MIT by Jesse Vincent.

Local preferences are proportionate fresh evidence, final-diff inspection,
explicit not-run and residual-risk reporting, and one semantic binary
conclusion by the main process. No fixed verdict token or structured state
record is required.
